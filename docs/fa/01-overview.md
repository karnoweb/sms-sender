# 01 — معرفی

## هدف پکیج

**SMS Sender** یک لایه‌ی انتزاعی سطح بالا برای ارسال پیامک در لاراول است که اپلیکیشن را از جزئیات سرویس‌دهنده‌های مختلف پیامک جدا می‌کند.

## ویژگی‌ها

- **رابط Fluent** — API زنجیره‌ای و خوانا
- **Failover خودکار** — سوئیچ خودکار بین درایورها
- **ثبت لاگ** — هر تلاش ارسال در دیتابیس ثبت می‌شود
- **Delivery Report** — بررسی وضعیت تحویل پیامک
- **قابل گسترش** — افزودن درایور جدید با پیاده‌سازی یک اینترفیس
- **کنترل مصرف** — محدودیت روزانه/ماهانه و فعال/غیرفعال کردن درایورها

## الزامات

- PHP 8.2+
- Laravel 11.x یا 12.x
